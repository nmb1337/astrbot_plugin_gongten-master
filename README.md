# astrbot_plugin_gongten —— 高危监控禁言

QQ 群高危监控禁言插件。管理员可将指定群成员加入监控名单，被监控用户一旦发言即自动禁言并发送警告。

## ✨ 功能

| 指令 | 权限 | 说明 |
|---|---|---|
| `/高危监控 @用户` 或 `/高危监控 QQ号` | 仅管理员 | 将用户加入高危监控名单 |
| `/联盟监控 @用户 [禁言秒数]` 或 `/联盟监控 QQ号 [禁言秒数]` | 仅管理员 | 将用户加入联盟监控；其后每条发言立即撤回并禁言 |
| `/联盟列表` | 所有人 | 查看当前群联盟监控名单及禁言秒数 |
| `/解除联盟监控 @用户` 或 `/解除联盟监控 QQ号` | 仅管理员 | 取消联盟监控并解除禁言 |
| `/高危列表` | 所有人 | 查看当前群监控名单（群名 + QQ号 + 昵称） |
| `/脱离监控 @用户` 或 `/脱离监控 QQ号` | 仅管理员 | 移出监控名单（支持QQ号，已退群也能解） |
| `/fin联盟 @用户` 或 `/fin联盟 QQ号` | 仅管理员 | 踢出群聊并加入黑名单（延迟3.5秒执行） |
| **自动禁言** | — | 被监控用户发言 → 秒禁言 + 警告 + 阻断AI响应 |
| **联盟监控** | — | 被监控用户发言 → 立刻撤回原消息 → 禁言指定秒数 + 阻断AI响应 |

## 🔧 更新记录 (v1.3.0)

- **联盟监控**：管理员可用 `/联盟监控 @用户 [禁言秒数]` 或 `/联盟监控 QQ号 [禁言秒数]` 监控成员；目标每次发言都会立即撤回并禁言。
- **单人时长设置**：命令末尾可设置该成员的禁言秒数；未设置时使用 `alliance_mute_duration` 默认值。
- **联盟名单管理**：新增 `/联盟列表` 和 `/解除联盟监控`，解除时会自动解禁。

## 🔧 历史更新 (v1.2.1)

- **脱离监控自动解禁**：`/脱离监控` 时自动调用 `set_group_ban duration=0` 解除禁言
- **双重防重复警告**：消息 ID 去重 + 同用户 8 秒内不重复禁言，杜绝刷屏
- **踢人先弹假消息**：`/fin联盟` 用 `asyncio.create_task` 后台踢人，`yield` 立刻发出"已被拉入联盟黑名单"，目标隔着 4 秒被踢也能看到消息
- **退群用户可解除**：支持 `/脱离监控 QQ号` 直接输入数字解除
- **踢人自动加黑名单**：`set_group_kick` 带 `reject_add_request=True`

## ⚙️ 配置项

在 AstrBot WebUI → 插件管理 → 本插件配置中可调整：

| 配置项 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `mute_duration` | int | 120 | 禁言时长（秒） |
| `alliance_mute_duration` | int | 120 | `/联盟监控` 未指定秒数时的默认禁言时长（1-2592000 秒） |
| `mute_warning` | string | 你已在高危监控名单，无法发送信息 | 禁言警告语 |

## 联盟监控使用

管理员可直接使用以下任一命令，均无需额外配置：

```text
/联盟监控 @用户
/联盟监控 @用户 600
/联盟监控 123456789 600
```

未填写秒数时，使用 `alliance_mute_duration`；填写秒数时按该成员单独保存。被联盟监控的成员每次发言都会先通过 OneBot `delete_msg` 撤回，再调用 `set_group_ban` 禁言。机器人需要拥有群管理员权限及撤回、禁言权限。

## 📦 安装

将本插件目录放入 AstrBot 的 `data/plugins/` 下，在 WebUI 中启用即可。

```bash
cd AstrBot/data/plugins
git clone https://github.com/yourname/astrbot_plugin_gongten.git
```
